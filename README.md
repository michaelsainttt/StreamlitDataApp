OVERVIEW
This Streamlit web application is to show the electric vehicle population data in an interactive way.
Users are able to explore trends in electric vehicle registration using charts

DATASET
The dataset used in this project is
**Electric Vehicle Population Data (CSV)**
Source: https://catalog.data.gov/dataset/electric-vehicle-population-data

FEATURES
Inteactive tabs for navigation
Data table view of the CSV file
Interactive Plotly chart

TECHNOLOGIES USED
Python
Streamlit
Pandas
Plotly
